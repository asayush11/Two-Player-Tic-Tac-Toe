# Two-Player-Tic-Tac-Toe ( Devleoped using SpringBoot - React ) ( https://tic-tac-toe-frontend-phi.vercel.app/ )

• It's a real-time multiplayer tic-tac-toe game developed using WebSocket.

• Users can invite their friends to play with them using invite links.

• System is scaled to support 200 concurrent games and 20000 active rooms within 0.1 CPU / 512 MB RAM with JVM tuning and Caffeine cache.

• Designed a responsive web interface using React with an average 325ms interaction latency for real-time gameplay.

• Configured webhooks in GitHub to trigger auto-build and deployment of dockerized service on Render and Vercel.
